# 个体仿真策略

## 个体类型

1. 个人/家户：基本消费单位。
（目前统一为个人）

2. 企业：基本生产单位。

3. 银行：储蓄、贷款、投资。
（是否需要区分普通银行和央行）

4. （可选）其他金融机构

5. （可选）政府：税收、转移支付、公共服务。

## 个体状态设计与决策模式

- 状态：该主体无法改变（环境状态或历史状态）且影响该主体决策的变量。

- 决策：该主体可以改变的变量。

- 限制：对决策的限制。

### 个人

#### 状态

1. 资产

    - 现金

    - 存款

2. 消费偏好

    - 给定效用函数和异质的参数。

3. 劳动能力

    - 基本能力

4. 教育水平

5. 冲击

    - 对劳动能力的冲击（个体劳动可以由劳动能力、教育水平和劳动冲击确定）

    - 对资产的冲击（风险资产）

#### 决策

1. 储蓄

    - 存款结构：现金/定期存款

2. 消费

    - 消费总量

    - 消费结构

3. 劳动

    - 找工作/换工作/离职

4. 教育

    - 选择是否接受教育

#### 限制

1. 资产不可为负

2. 最小消费（基本生活需求）

### 企业

#### 状态

1. 资产

    - 现金

    - 存款

    - 库存

2. 生产函数

    - 给定生产函数和异质的参数。

3. 冲击

#### 决策

1. 资产

    - 投资：投资本企业或其他企业

    - 贷款

    - 处置现有库存

2. 生产

    - 决定生产规模、要素分配策略

        - 雇佣/解雇员工

    - 定价

#### 限制

1. 总资产不可为负。

2. 最小生产规模：基本工资。

### 央行

#### 状态

1. 宏观经济指标

    - 通货膨胀率

    - 失业率

    - GDP 产出缺口

    - 信贷总量

2. 政策目标

    - 通胀目标

    - 就业目标

3. 资产负债表

    - 资产：政府债券、外汇储备、对商业银行的再贷款

    - 负债：基础货币（流通中的货币、商业银行存款准备金）

#### 决策

1. 法定准备金率

2. 基准利率

3. 公开市场操作

#### 限制（软约束）

1. 政策使命：维持物价稳定、促进经济增长

2. 维护金融系统稳定：不能采取可能引发系统性金融风险的极端政策。在必要时，需承担最后贷款人的责任。

### 商业银行

#### 状态

1. 资产负债表

2. 财务健康指标：实际准备金率、风险与流动性

3. 外部市场环境

    - 法定准备金率和基准利率

    - 市场无风险利率

#### 决策

1. 存款利率和贷款利率

2. 信贷审批

3. 资产负债管理

#### 限制

1. 法定准备金率要求

2. 资本充足率要求

3. 流动性要求

### （可选）其他金融机构

### （可选）政府

#### 状态

1. 财政状况

    - 财政收入：税收总额

    - 财政支出：政府购买、转移支付、公务员工资、国债利息支出

    - 财政赤字/盈余

2. 债务状况

    - 国债存量

    - 国债利率

3. 宏观经济感知

    - GDP、失业率、通胀率、基尼系数

4. 政策目标

    - 经济增长目标

    - 充分就业目标

    - 社会公平目标

#### 决策

1. 税收政策：个人所得税率、企业所得税率、消费税率

2. 财政支出：政府购买水平、转移支付规模、公共部门雇佣规模

3. 发行国债

#### 限制

1. 预算约束

    - 当期支出 = 当期税收 + 当期新增国债

2. （软约束）债务可持续性/国债上限

3. （软约束）政治与社会约束
